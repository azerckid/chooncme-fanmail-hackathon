## 1. 환경변수 수정 (오늘 밤, 필수)

### 1-1. CRON_SECRET 추가

`.env.local`에서 현재 주석 처리됨. cron 엔드포인트(`/api/cron/check-email`) 인증에 필수.

```
# 변경 전
# CRON_SECRET=

# 변경 후
CRON_SECRET=chooncme-cron-2026
```

### 1-2. DEMO_MODE 활성화

지연 발송을 10~30분 → 0~30초로 단축. 심사 중 실시간 데모에 필수.

```
# 변경 전
DEMO_MODE=false

# 변경 후
DEMO_MODE=true
```

### 1-3. 행사 당일 오전 — Flock.io 크레딧 수령 후 확인

크레딧은 행사 당일 오전 azerckid@gmail.com으로 지급 예정.
현재 `FLOCK_API_KEY`가 세팅되어 있으나, 크레딧 지급 후 플랫폼에서 키 유효성 재확인.

- [ ] https://www.flock.io/platform 로그인 → API Key 상태 확인
- [ ] Nansen(https://nansen.ai/) 로그인 → 크레딧 수령 확인

## 6. 데모 시나리오 (발표 순서)

심사 시간: 17:10~ / 팀당 약 3~5분 예상

### Step 1 — 대시보드 소개 (30초)
- `http://localhost:3000` 대시보드 화면
- 수신된 팬 레터 목록, 답장 현황 보여주기

### Step 2 — 파이프라인 라이브 데모 (1분 30초)
1. `choon.cme@gmail.com`으로 팬메일 직접 발송 (미리 준비)
2. 대시보드에서 수신 → 분류(Flock.io LLM) → 답장 생성 실시간 확인
3. DEMO_MODE=true 상태이므로 30초 내 답장 발송
4. 발신 Gmail 계정에서 답장 수신 확인

### Step 3 — Base 온체인 기록 보여주기 (1분)
- Blockscout에서 NFT 민팅 트랜잭션 확인
- "팬레터 답장이 Base에 영구 기록됩니다" 강조

### Step 4 — 기술 스택 한 줄 요약 (30초)
- Flock.io LLM → 춘심이 페르소나 답장 생성
- AgentKit → Base 지갑으로 자율 운용
- x402 → LLM 비용 자율 결제 (활성화 시)
- Virtuals ACP → 멀티 에이전트 협업
