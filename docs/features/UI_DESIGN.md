# UI/UX 설계서 (UI/UX Design Specification)

## 1. 디자인 컨셉 (Design Concept)
- **테마:** 춘심이 캐릭터의 친근하고 구수한 느낌을 살리되, 관리 도구로서의 깔끔함을 유지.
- **컬러 팔레트 (Neutral & Warm):**
  - **Primary:** 따뜻한 흙색 또는 짙은 베이지 계열 (Neutral 800~900)
  - **Background:** 종이 질감이 느껴지는 미색 (Neutral 50)
  - **Accent:** 춘심이 스카프 색상 (Optional)
- **타이포그래피:** 가독성이 좋은 산세리프 (`Inter`)를 기본으로 하되, 헤드라인에는 약간의 레트로한 느낌을 줄 수 있음.

## 2. 화면 구성 (Sitemap)

### 2.1 메인 대시보드 (Dashboard Home) - `/`
- **목표:** 전체적인 현황 파악 및 최신 팬레터 확인.
- **주요 컴포넌트:**
  - **통계 카드:** 총 받은 편지 수, 이번 주 받은 편지, 평균 답장 길이.
  - **최근 활동 피드:** 가장 최근에 오고 간 대화 쌍(Pair) 리스트.

### 2.2 팬레터 목록 (Letter List) - `/letters`
- **목표:** 저장된 모든 팬레터를 검색하고 필터링.
- **기능:**
  - **검색창:** 내용, 닉네임으로 검색.
  - **테이블/카드 뷰:** 편지 도착 시간, 보낸 사람, 답장 여부 표시.
  - **상태 배지:** `답장 완료`, `분석됨` 등의 태그 표시.

### 2.3 대화 상세 뷰 (Conversation Detail) - `/letters/[id]`
- **목표:** 특정 팬레터와 그에 대한 답장을 하나의 맥락에서 열람.
- **레이아웃:** 채팅방(Message Bubble) 스타일 UI.
  - **좌측:** 팬의 메시지 (Letter)
  - **우측:** 춘심이의 답장 (Reply)
- **메타데이터 패널:** 화면 우측 사이드바에 감정 점수, 모델 버전, 타임스탬프 표시.

## 3. 주요 인터랙션 (Key Interactions)
- **Hover Effects:** 리스트 항목에 마우스를 올리면 은은한 배경색 변화.
- **Copy to Clipboard:** 편지나 답장 내용을 클릭 한 번으로 복사.
- **Responsive:** 모바일에서도 편지 내용을 쉽게 읽을 수 있도록 카드형 레이아웃 변환.
